Ocorrências

As ocorrências são uma parte fundamental do sistema. Sempre quando um equipamento chega para o conserto, é necessário abrir uma ocorrência, informando o problema, o cliente e outros dados relevantes;

Para cadastrar uma ocorrência: Menu > Cadastro > Ocorrências;

Para consultar ou alterar uma ocorrência: Menu > Consulta > Ocorrências, ou se ela estiver aberta ela pode ser localizada na tabela de ocorrências em aberto na tela principal do sistema;
