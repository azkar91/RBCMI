Tela Principal

A janela principal é o local onde temos o acesso a todoas as funcionalidades do sistema;

O menu principal contém todos os itens e janelas acessíveis: Arquivo, Cadastro, Consulta e Ajuda;

O item de menu Arquivo possui um subitem Sair, por onde é possível finalizar a execução do sistema;

O item de menu Cadastro é o local onde encontram-se todas as telas de cadastro de informações do sistema, que são: Clientes, Técnicos, Ocorrências e Casos;

O item de menu Consulta é o item que dá acesso a todas as informações armazenadas no sistema através das telas de cadastros, além de possibilitar a edição das mesmas;

O item de menu Ajuda é o local que contém uma breve documentação do sistema com o objetivo de auxilar o usuário na operação do sistema;

Ainda na tela principal do sistema está localizada uma tabela que dá acesso rápido a todas ocorrências em aberto, facilitando a vizualização do usuário.
