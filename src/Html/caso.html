Casos

O Raciocínio Baseado em Casos é uma técnica que utiliza de casos para recuperar e armazenar conhecimento oriundos de experiências. O caso contém os atributos que o identificam e informações que possibilitam o sistema aprender e adaptar uma solução para um novo problema.

Para cadastrar um caso separadamente: Menu > Cadastro > Caso;

Outra opção interessante é cadastrar um caso junto a ocorrência (visto que isso é necessário), para isto, abra uma ocorrência e clique no botão Adicionar Caso na janela de atualização de Ocorrências;

Os casos podem ser cadastrados de duas formas, por Categoria de Poblema (recomendável) ou por Seleção Livre de Atributos (quando os atributos não se encaixam em somente uma categoria);

Depois de preencher todos os campos, o clico RBC é iniciado, durante o processo, para qualquer dúvida, é recomedável movimentar o mouse sobre os campos para obter ajuda;

Para consultar um caso: Menu > Consulta > Casos.
